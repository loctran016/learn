## Document structure tags

```html
<html>
    <head>
        <title>Page title</title>
    </head>
    <body>
    <h1>Welcome to My Website</h1>
    <p>This is a paragraph introducing the content of my website.</p>
    <a href="https://www.example.com">Visit Example.com</a>
    </body>
</html>
```

## Commonly used tags

```html
<div>Block element</div>

<p>Block paragraph</p>

<span>Incline Text</span>

<a href="..." target="_blank">...</a>

<img src="..." alt="..." />

<iframe src="..."></iframe>

<ul>
    <li>Unordered Lists</li>
    <li>JS</li>
    <li>CSS</li>
    <li>HTML</li>
</ul>

<ol>
    <li>Ordered Lists</li>
    <li>One</li>
    <li>Two</li>
    <li>Three</li>
</ol>

<table>
    <th>
        <td>Rows/Columns</td>
        <td>First column</td>
        <td>Second column</td>
    </th>

    </tr>
    <tr>
        <td>Second row</td>

</table>
```
